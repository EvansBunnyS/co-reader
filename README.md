# 📖 Co-Reader v3 — SillyTavern 共读批注扩展

将任何文章拖入 SillyTavern，与 AI 角色卡共读、批注、多轮对话。

---

## ✨ v3 全功能一览

### 📚 书架 & 封面
| 功能 | 说明 |
|------|------|
| 多书管理 | 无限导入，书架卡片网格展示 |
| 自定义封面 | 悬停书卡 → 点击 🖼 上传任意图片，存为 base64 |
| 阅读时间戳 | 自动记录最后打开时间，按此排序 |
| 删除书籍 | 悬停书卡右上角 🗑 |

### ✏️ 批注
| 功能 | 说明 |
|------|------|
| 用户批注 | 选中文字 → 弹窗 → 💬 我来批注 |
| AI 批注  | 选中文字 → 🤖 AI批注（可附带提示语）|
| AI 全文批注 | 工具栏 🤖 全文批注，自动提取开头/中段/结尾精华 |
| 批注导入 | 导入之前导出的 JSON，合并到当前书 |
| 批注链接 | 卡片 🔗 → 选择关联的其他批注，形成批注网络 |
| 筛选搜索 | 按来源（用户/AI选批/AI全文）+ 关键词搜索 |
| 三色高亮 | 紫=用户批注 · 青=AI选批 · 绿=AI全文批注 |

### 🔁 多轮对话
- 每条批注下方有无限轮对话框
- AI 回复时自动注入**角色卡**名称与性格
- 「自动AI回复」开关：批注/回复后立即触发

### 🔍 字词解释 & 🌐 翻译
- 选中 **≤30字** 的短文本时，弹窗额外出现：
  - **🔍 解释**：词义、用法、典故（角色口吻作答）
  - **🌐 翻译**：目标语言可在设置中自定义（默认中文）
- 结果可**存为批注**或**复制**

### 📊 进度条 & 书签
- 顶部细条实时显示阅读百分比
- 书签段落在进度条上显示为**紫点**，点击快速跳转
- 长按段落（手机）→ 添加/取消书签
- 双击段落（桌面）→ TTS 朗读（需开启 TTS）

### 🔊 TTS 朗读
- 设置中开启 TTS
- 双击正文段落 → 朗读整段
- 弹窗 🔊 → 朗读选中片段
- 批注卡片 🔊 → 朗读批注内容
- 语速可在设置拖动调节

### 🎭 共读模式
- 工具栏开启后，AI 在**正常聊天**时也能引用书中内容
- 自动将书名、阅读进度、近期批注注入到角色卡上下文
- 随时关闭，不污染对话历史

### 🧠 导出角色记忆
- 工具栏 🧠 → 提取全书批注摘要
- 自动**复制到剪贴板** + 弹窗显示完整内容
- 粘贴进角色卡「记忆」或「世界信息」即可长期保留

### 🎨 四套主题包
| 主题 | 风格 |
|------|------|
| 深夜猫咖 | Catppuccin 深紫，默认 |
| 赛博霓虹 | 极暗背景 + 荧光青色 |
| 日系米白 | 护眼暖白，适合白天 |
| 复古牛皮纸 | 暖棕系，仿纸质感 |

主题可叠加手动调色（背景/文字/强调色独立修改）。

## 🚀 安装

### 方式一：ST 扩展管理器（推荐）
1. SillyTavern → 插头图标 → Install Extension
2. 填入仓库地址 → Install → 刷新

### 方式二：手动
```bash
cd SillyTavern/public/scripts/extensions/third-party/
git clone https://github.com/你的用户名/co-reader
```
刷新页面即生效。

## 📱 手机操作速查

| 操作 | 方式 |
|------|------|
| 打开/关闭 | 点击右下角 📖 |
| 移动悬浮球 | 按住拖动 |

---

## 💾 数据安全

- 所有数据（书本内容、批注、封面）存储于
  `SillyTavern/data/default-user/settings.json`
  `extension_settings["co-reader"]` 字段
- **重装扩展不丢失数据**
- 建议定期用 ⬇ JSON 备份
